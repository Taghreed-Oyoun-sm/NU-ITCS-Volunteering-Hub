# NU ITCS Volunteering Hub – Full Requirements

## Functional Requirements

| ID  | Requirement                            | Description                                                 |
|-----|----------------------------------------|-------------------------------------------------------------|
| FR1 | User Registration & Login              | Secure signup/login with email & password                   |
| FR2 | Post Creation                          | Create academic help requests or volunteering opportunities |
| FR3 | Notifications                          | Alert suitable volunteers when help is needed               |
| FR4 | Search & Tagging                       | Posts tagged with #DSA, #MathForAI, #EventVolunteer, etc.   |
| FR5 | Endorsement System                     | TAs/Dr can endorse students (like LinkedIn recommendations) |
| FR6 | Admin Moderation                       | Admins approve, Deny and delete posts and comments          |
| FR7 | Report System                          | Users can report posts/comments to admin                    |


## Volunteer Criteria System (Core Feature)
When a student signs up → must create profile with:  
- CGPA  
- Strength Areas (e.g., DSA, OOP, Logic Design, Math)  
- Skills & Interests  

## Non-Functional Requirements
- **Usability:** Simple, mobile-friendly interface  
- **Security:** Password hashing, JWT auth, protected routes  
- **Performance:** Fast loading even with many posts  
- **Scalability:** Support future expansion to other departments  
- **Availability:** Accessible 24/7 via university portal or direct link  
- **Maintainability:** Clean code, documentation, easy admin updates

## System Users & Roles
| Role                | Permissions |
|---------------------|-----------|
| Student             | Post, request help, volunteer, get endorsed |
| TA and DR           | Endorse students, post research/volunteering ops |
| Club Representative | Post event volunteering opportunities |
| Admin (Moderator)   | Approve posts, delete content, manage reports |
